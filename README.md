# Eye-Movement-Dataset-BIT.IFE-

## Overview

This repository contains a limited, low-resolution subset of processed eye-region images derived from an eye-tracking study, and is used in the article &lt;Symmetry-reconfigurable photodiodes for readout circuitry-integrated transmissive imaging and neuromorphic eye-machine interfacing>

All images in this repository:

- Contain only cropped eye-region areas
- Are downsampled to very low resolution
- Have undergone hardware-level processing
- Do not contain facial regions
- Do not contain direct personal identifiers

The released subset is intended for reproducibility verification and methodological reference only.

---

## Data Description

This repository does NOT include:

- Raw video recordings
- High-resolution images
- Full experimental data
- Any direct personal identifiers

Only a limited processed subset is publicly available.
